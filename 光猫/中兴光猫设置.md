# 查询和修改区域代码（操作前请拔掉光纤）
查询区域代码
```
cat /etc/init.d/regioncode 
```
修改区域代码
```
upgradetest sdefconf XXX
```
修改完毕后光猫会自动重启并且会关闭，telnet需要再次开启
